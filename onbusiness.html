香港濟歲堂出差管理制度

深圳团队适用版 2025年9月修订

一、总则

为规范深圳团队员工出差管理，合理控制差旅费用，提高出差效率，保障员工权益，特制定本制度。

适用范围：本制度适用于香港济岁堂深圳团队所有因公出差的员工。

团队成员居住地：深圳市

临时办事处：龙岗区

主营出差地：香港、全国范围

二、出差类型与目的

香港出差 国内出差
  • • 在建工厂或业务
  • • 中转牌照批次手续办理
  • • 政府部门沟通协调
  • • 本地供应商考察
  • • OEM合作厂商寻访
  • • 原料供应商考察
  • • 行业会议参与
  • • 合作伙伴洽谈
  • • 分公司业务支持

四、简化出差审批流程

简化审批流程（10人以下团队适用）
  • 员工在飞书提交《出差申请单》
  • 直接主管1个工作日内审批
  • 审批通过后行政部备案并安排差旅事宜
  • 财务部根据审批结果预支差旅费用（如需要）

特别提醒：

由于团队成员居住在深圳，前往香港出差需提前办理港澳通行证及商务签注，请至少提前6个工作日申请。

五、深圳至香港出差安排

交通方式

  • • 优先选择地铁/港铁出行（深圳地铁至口岸，转乘港铁）
  • • 可选择跨城巴士或高铁（深圳北/福田站至香港西九龙）
  • • 特殊情况可申请公司车辆接送（需提前审批）

住宿安排

  • • 原则上当日往返，减少住宿开支
  • • 如因工作需要住宿，需在申请时说明理由
  • • 跟领导出发：香港住宿标准港币950元/晚以内
  • • 无领导陪同：香港住宿标准港币800元/晚以内

六、出差期间工作要求

1. 跟领导出发

  • • 提前了解出差目的和行程安排
  • • 准备相关业务资料和汇报材料
  • • 协助领导处理日常事务
  • • 重要会议做好记录并整理会议纪要
  • • 遵守公司章程协议，不得泄露公司机密信息

2. 无领导陪同

  • • 保持与公司的日常沟通，每日汇报工作进展
  • • 重要会议或会谈结束后记录并发送回公司
  • • 独立处理出差相关业务事项
  • • 遵守公司章程协议，不得泄露公司机密信息
  • • 注意人员和财产安全，购买必要的旅行保险

所有出差人员均需遵守当地法律法规及文化习俗。

七、出差归来后续工作

  • • 3个工作日内提交出差报告，包括工作成果、遇到的问题及建议
  • • 7个工作日内完成费用报销，附上所有票据及审批文件
  • • 根据需要组织出差成果分享会
  • • 提交出差期间达成的合作协议或意向

八、常用路线参考（龙岗办事处至香港）

  • • 地铁：3号线至福田口岸 → 港铁东铁线
  • • 高铁：深圳北站 → 香港西九龙站（约30分钟）
  • • 跨城巴士：龙岗城市候机楼 → 香港各区

九、附则

  • • 本制度自发布之日起执行
  • • 本制度由行政部负责解释和修订
  • • 特殊情况可根据实际需求经总经理批准后调整执行
