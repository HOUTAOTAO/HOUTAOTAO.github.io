中成药制造牌照申请

根据香港《中医药条例》，任何人士欲在香港制造中成药作销售用途，必须向中医药规管办公室申请中成药制造牌照。

核心法规要求

  • 《中医药条例》（第549章）第111-123条
  • 《中成药生产质量管理规范》（GMP）
  • 《药剂业及毒药条例》（第138章）相关要求

申请资格

申请人要求

申请人须为香港注册公司或年满18岁的香港居民

厂房要求

厂房须符合GMP要求并通过现场检查

人员要求

须指定一名合资格人士担任负责药师

申请文件清单

  • 填妥的申请表格（表格CM-1）
  • 公司注册证明书（副本）
  • 厂房平面图、设备清单及照片
  • 生产质量管理文件（GMP手册）
  • 负责药师的资格证明及聘用文件
  • 产品配方、制造流程及质量控制程序
  • 环境评估报告（如适用）
  • 厂房租赁或所有权证明
  • 产品稳定性测试报告

申请流程

1. 前期准备

确保厂房符合GMP标准，准备所有质量管理文件，任命负责药师

2. 提交申请

向中医药规管办公室提交完整申请文件及申请费用 HK$7,250

3. 文件审核

办公室审核提交文件，可能要求补充资料（约1-2个月）

4. 现场检查

GMP合规检查小组进行厂房现场核查（约1-3个月）

5. 审批决定

中医药管理委员会审议申请（约1-2个月）

6. 牌照签发

获批后签发牌照，有效期通常为3年（可续期）

申请时间周期

1
准备阶段：2-4个月

厂房建设/改造，文件准备，人员培训

2
申请处理：4-8个月

从提交申请到最终决定

3
总时间：6-12个月

取决于厂房准备情况和文件完整性

重要注意事项

  • 制造牌照仅允许生产已注册的中成药产品
  • 牌照持有人需定期接受GMP复检（通常每2-3年一次）
  • 任何厂房设施变更需提前向办公室申报
  • 牌照续期需在到期前3个月提出申请
  • 需建立完善的质量管理体系和产品追溯系统

中成药外包装牌照申请

根据香港《中医药条例》，从事中成药外包装业务需单独申请牌照，确保包装过程符合GMP要求。

核心法规要求

  • 《中医药条例》（第549章）第111-123条
  • 《中成药生产质量管理规范》（GMP）包装章节
  • 《中成药标签指引》

申请资格

申请人要求

香港注册公司或年满18岁的香港居民

场地要求

专用包装区域符合GMP要求

人员要求

具备包装操作培训合格人员

申请文件清单

  • 填妥的申请表格（表格CM-P1）
  • 商业登记证（副本）
  • 包装车间平面图及设备清单
  • 包装质量管理文件
  • 包装流程标准操作规程（SOP）
  • 标签设计样本及审核文件
  • 包装材料供应商资质文件
  • 场所租赁或所有权证明
  • 防伪防窜货措施说明

申请流程

1. 前期准备

建立包装质量管理体系，准备包装SOP，培训员工

2. 提交申请

提交完整申请文件及申请费用 HK$5,800

3. 文件审核

办公室审核提交文件（约1-2个月）

4. 现场检查

GMP合规检查小组进行现场核查（约1-2个月）

5. 审批决定

委员会审议申请（约1个月）

6. 牌照签发

获批后签发牌照，有效期3年

申请时间周期

1
准备阶段：1-3个月

场地设置，文件准备，人员培训

2
申请处理：3-5个月

从提交申请到最终决定

3
总时间：4-8个月

取决于场地准备情况和文件完整性

重要注意事项

  • 只能为已注册的中成药产品提供包装服务
  • 包装材料需符合药用标准并提供检验报告
  • 标签内容必须符合《中成药标签指引》要求
  • 需建立完善的批包装记录系统
  • 包装过程需防止交叉污染和混淆

中成药批发牌照申请

根据香港《中医药条例》，任何人士欲在香港批发中成药，必须向中医药规管办公室申请中成药批发牌照。

核心法规要求

  • 《中医药条例》（第549章）第129-135条
  • 《中成药批发质量管理指引》
  • 《进出口条例》（第60章）相关要求

申请资格

申请人要求

香港注册公司或年满18岁的香港居民

仓库要求

须具备符合要求的储存设施

系统要求

须建立完善的库存管理系统

申请文件清单

  • 填妥的申请表格（表格CM-2）
  • 商业登记证（副本）
  • 仓库平面图及储存设施说明
  • 质量管理系统文件
  • 人员资质证明
  • 供应商及客户管理程序
  • 场所租赁或所有权证明
  • 温湿度监控系统说明
  • 产品追溯系统说明

申请流程

1. 前期准备

确保仓库符合储存要求，建立质量管理体系

2. 提交申请

提交完整申请文件及申请费用 HK$3,600

3. 文件审核

办公室审核提交文件（约1个月）

4. 现场检查

检查小组进行仓库现场核查（约2-4周）

5. 审批决定

委员会审议申请（约1个月）

6. 牌照签发

获批后签发牌照，有效期3年

申请时间周期

1
准备阶段：1-2个月

仓库设置，文件准备，系统建立

2
申请处理：3-5个月

从提交申请到最终决定

3
总时间：4-7个月

取决于仓库准备情况和文件完整性

重要注意事项

  • 只能批发已注册的中成药产品
  • 必须建立完整的药品追溯系统
  • 储存环境需符合产品标签要求（温湿度控制等）
  • 每年需提交业务活动报告
  • 进口中成药需额外申请进口许可证

中成药零售牌照申请

根据香港《中医药条例》，任何零售店欲销售中成药，必须向中医药规管办公室申请中成药零售牌照。

核心法规要求

  • 《中医药条例》（第549章）第119-128条
  • 《中成药标签指引》
  • 《不良医药广告条例》（第231章）

申请资格

申请人要求

香港注册公司或年满18岁的香港居民

场所要求

零售场所符合卫生及储存要求

人员要求

负责人需完成认可中成药课程

申请文件清单

  • 填妥的申请表格（表格CM-3）
  • 商业登记证（副本）
  • 店铺平面图及照片
  • 负责人身份证明及资格证书
  • 产品储存安排说明
  • 店铺租赁或所有权证明
  • 中成药供应商名单
  • 产品退货处理程序

申请流程

1. 前期准备

确保店铺符合要求，完成必要培训

2. 提交申请

提交完整申请文件及申请费用 HK$1,200

3. 文件审核

办公室审核提交文件（约2-3周）

4. 现场检查

检查小组进行零售场所核查（约1-2周）

5. 审批决定

委员会审议申请（约2-3周）

6. 牌照签发

获批后签发牌照，有效期3年

申请时间周期

1
准备阶段：1-2个月

店铺准备，人员培训，文件整理

2
申请处理：2-3个月

从提交申请到最终决定

3
总时间：3-5个月

常规零售牌照申请通常较快速

重要注意事项

  • 只能销售已注册的中成药产品
  • 需在显眼处展示牌照正本
  • 遵守中成药广告宣传限制
  • 不得销售处方中成药（除非持相关许可）
  • 需妥善处理顾客查询及投诉

共同要求

产品要求

只能处理已注册的中成药产品

牌照续期

需在到期前3个月提出续期申请

变更申报

重要变更需及时向办公室申报

牌照申请通用原则

  • 所有申请均需通过中医药规管办公室审批
  • 牌照不可转让，仅限申请场所使用
  • 需建立产品追溯系统
  • 定期接受卫生署的监管检查
  • 需保存完整记录至少2年
  • 牌照持有人对运营负全部法律责任
