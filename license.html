中成药制造牌照申请

根据香港《中医药条例》，任何人士欲在香港制造中成药作销售用途，必须向中医药规管办公室申请中成药制造牌照。

核心法规要求

  • 《中医药条例》（第549章）第111-123条
  • 《中成药生产质量管理规范》（GMP）
  • 《药剂业及毒药条例》（第138章）相关要求（如适用）

申请资格

  • 申请人须为香港注册公司或年满18岁的香港居民
  • 厂房须符合GMP要求并通过现场检查
  • 须指定一名合资格人士担任负责药师

申请文件清单

  • 填妥的申请表格（表格CM-1）
  • 公司注册证明书（副本）
  • 厂房平面图及设备清单
  • 生产质量管理文件（GMP手册）
  • 负责药师的资格证明及聘用文件
  • 产品配方及制造流程说明
  • 环境评估报告（如适用）
  • 厂房租赁或所有权证明

📋 1. 前期准备

确保厂房符合GMP标准，准备所有质量管理文件，任命负责药师。

📬 2. 提交申请

向中医药规管办公室提交完整申请文件及申请费用（目前为港币7,250元）。

🔍 3. 文件审核

办公室审核提交文件，可能要求补充资料（约1-2个月）。

🏭 4. 现场检查

GMP合规检查小组进行厂房现场核查（约1-3个月）。

✅ 5. 审批决定

中医药管理委员会审议申请（约1-2个月）。

📄 6. 牌照签发

获批后签发牌照，有效期通常为3年（可续期）。

申请时间周期

准备阶段：2-4个月

厂房建设/改造，文件准备，人员培训

申请处理：4-8个月

从提交申请到最终决定

总时间：6-12个月

取决于厂房准备情况和文件完整性

重要注意事项

  • 制造牌照仅允许生产已注册的中成药产品
  • 牌照持有人需定期接受GMP复检（通常每2-3年一次）
  • 任何厂房设施变更需提前向办公室申报
  • 牌照续期需在到期前3个月提出申请

中成药批发牌照申请

根据香港《中医药条例》，任何人士欲在香港批发中成药，必须向中医药规管办公室申请中成药批发牌照。

核心法规要求

  • 《中医药条例》（第549章）第129-135条
  • 《中成药批发质量管理指引》
  • 《进出口条例》（第60章）相关要求

申请资格

  • 申请人须为香港注册公司或年满18岁的香港居民
  • 须具备符合要求的储存设施
  • 须建立完善的库存管理系统

申请文件清单

  • 填妥的申请表格（表格CM-2）
  • 商业登记证（副本）
  • 仓库平面图及储存设施说明
  • 质量管理系统文件
  • 人员资质证明
  • 供应商及客户管理程序
  • 场所租赁或所有权证明

📋 1. 前期准备

确保仓库符合储存要求，建立质量管理体系，任命负责人。

📬 2. 提交申请

向中医药规管办公室提交完整申请文件及申请费用（目前为港币3,600元）。

🔍 3. 文件审核

办公室审核提交文件，可能要求补充资料（约1个月）。

🏢 4. 现场检查

检查小组进行仓库现场核查（约2-4周）。

✅ 5. 审批决定

中医药管理委员会审议申请（约1个月）。

📄 6. 牌照签发

获批后签发牌照，有效期通常为3年（可续期）。

申请时间周期

准备阶段：1-2个月

仓库设置，文件准备，系统建立

申请处理：3-5个月

从提交申请到最终决定

总时间：4-7个月

取决于仓库准备情况和文件完整性

重要注意事项

  • 只能批发已注册的中成药产品
  • 必须建立完整的药品追溯系统
  • 储存环境需符合产品标签要求（温湿度控制等）
  • 每年需提交业务活动报告

中成药零售牌照申请

根据香港《中医药条例》，任何零售店欲销售中成药，必须向中医药规管办公室申请中成药零售牌照。

核心法规要求

  • 《中医药条例》（第549章）第119-128条
  • 《中成药标签指引》
  • 《不良医药广告条例》（第231章）

申请资格

  • 申请人须为香港注册公司或年满18岁的香港居民
  • 零售场所符合基本卫生及储存要求
  • 负责人需完成认可中成药课程（如适用）

申请文件清单

  • 填妥的申请表格（表格CM-3）
  • 商业登记证（副本）
  • 店铺平面图及照片
  • 负责人身份证明及资格证书
  • 产品储存安排说明
  • 店铺租赁或所有权证明

📋 1. 前期准备

确保店铺符合要求，完成必要培训，准备所需文件。

📬 2. 提交申请

向中医药规管办公室提交完整申请文件及申请费用（目前为港币1,200元）。

🔍 3. 文件审核

办公室审核提交文件（约2-3周）。

🏪 4. 现场检查

检查小组进行零售场所核查（约1-2周）。

✅ 5. 审批决定

中医药管理委员会审议申请（约2-3周）。

📄 6. 牌照签发

获批后签发牌照，有效期通常为3年（可续期）。

申请时间周期

准备阶段：1-2个月

店铺准备，人员培训，文件整理

申请处理：2-3个月

从提交申请到最终决定

总时间：3-5个月

常规零售牌照申请通常较快速

重要注意事项

  • 只能销售已注册的中成药产品
  • 需在显眼处展示牌照正本
  • 遵守中成药广告宣传限制
  • 不得销售处方中成药（除非持相关许可）
